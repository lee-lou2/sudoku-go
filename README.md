# Golang Sudoku Solution

고를 이용해서 스도쿠를 빠르고 정확하게 해결할 수 있는 솔루션 어플리케이션

## Skill

- Lang : Golang

## Example

- website : [sudoku.lou2.kr](sudoku.lou2.kr)

## Install

1. git clone or fork
```bash
git clone https://github.com/lee-lou2/sudoku-go.git
```

2. install golang modules
```bash
go mod tidy
```

3. run
```bash
go run .
```
